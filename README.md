# Proposition 50 Voting Analysis

A comprehensive web application analyzing California Proposition 50's impact on congressional redistricting.

## Overview

Proposition 50, officially titled the "Election Rigging Response Act," is a proposed amendment to the California Constitution appearing on the special election ballot on November 4, 2025. This measure seeks to authorize temporary redrawing of California's congressional district maps in response to partisan redistricting efforts in other states.

## Features

- **Comprehensive Overview**: Detailed explanation of Proposition 50's purpose, timeline, and fiscal impact
- **District-by-District Analysis**: In-depth breakdown of how each affected district would change
- **Interactive Visualizations**: Charts and graphs showing political impact, vote share changes, and competitiveness
- **Impact Analysis**: Analysis of political, geographic, demographic, and legal impacts
- **Data Tables**: Sortable tables with detailed district information
- **Filtering**: Filter districts by party affiliation or impact level
- **Sources & Citations**: Comprehensive list of sources with links to official documents and news coverage

## Usage

Simply open `index.html` in a modern web browser. No build process or server required - this is a static web application.
